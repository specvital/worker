# Specvital 제품 개요

> 🇺🇸 [English Version](/en/prd/00-overview.md)

> 소스코드 정적 분석 기반 테스트 인벤토리 자동 생성 플랫폼

## 핵심 가치

**"GitHub URL만 입력하면 즉시 전체 테스트 현황 파악"**

- **즉시성**: CI/CD 없이 즉시 분석
- **포괄성**: 주요 언어/프레임워크 지원
- **정확성**: Tree-sitter AST 기반 파싱

## 타겟 사용자

1. **Engineering Manager**: 팀 테스트 현황 파악
2. **QA Lead**: 자동화 테스트 인벤토리 관리
3. **Staff Engineer**: 레거시 리팩토링 전 테스트 맵

## 경쟁 포지션

| 경쟁사      | 접근법    | Specvital 차별점 |
| ----------- | --------- | ---------------- |
| TestRail    | 수동 입력 | 자동 추출        |
| Qase        | AI 지원   | 정적 분석        |
| Testomat.io | JS 특화   | 다중 언어        |
